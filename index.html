Front-end

Jak wczesniej wspomniałem ze moje hobby to webdeveloping, jednak jest to tak obszerny temat i tym co sie interesuje i ulepszam swoje mozliwosci to jest front-end. Jest to mozna powiedzieć "odłam" który zajmuje sie głównie wyglądem strony i jej interfejsem.

Głównymi jezykami których sa 3 należą:

HTML
CSS
JavaScript

HTML (Hypertext Markup Language) jest to kod używany do tworzenia struktury strony i jej zawartości. Na przykład treść może być uporządkowana w obrębie zestawu akapitów, listy punktowanych punktów lub tabel obrazów i danych. Jak sugeruje tytuł, ten artykuł daje podstawy do zrozumienia HTML i jego funkcji.s

HTML nie jest językiem programowania; jest językiem znaczników (ang. markup language). HTML składa się z serii znaczników (tagów), których używa się do zamknięcia, opakowania różnych części treści, tak aby wyglądały i/lub działały w określony sposób.

Kaskadowe arkusze stylów (CSS) są językiem stylów używanym do opisania warstwy prezentacji dokumentów napisanych w HTML lub XML (włączając pochodne SVG czy XHTML). CSS opisuje jak elementy powinny być prezentowane na ekranie, przy wydruku, w czytniku ekranowym, albo w innych mediach

CSS jest jednym z podstawowych języków otwartej sieci i jest ustandaryzowany pomiędzy przeglądarkami zgodnie ze specyfikacją W3C. Rozwijany stopniowo, CSS1 obecnie przestarzały , CSS2.1 rekomendowany, oraz CSS3, obecnie rozbity na mniejsze moduły w ciągłym procesie standaryzacji.

JavaScript to język programowania, który umożliwia wdrożenie na stronie internetowej skomplikowanych elementów, dzięki którym strona ta może nie tylko wyświetlać statyczne informacje, ale również obsługiwać zmianę treść odpowiednio do sytuacji, wyświetlać interaktywne mapy i animacje grafiki 2D/3D , wyświetlać video itd. Jest to trzecia warstwa standardowych technologii internetowych.

Żródło: https://developer.mozilla.org

Oczywiście wraz z rozwojem front-endu powstały rożne techonologi pisania w tych jezykach. Przykładami mogą byc preprocesory, które pozwalają nam pisac kaskadowe style o wiele wiele szybciej oraz nadaja temu jezykowi opisowemu troche elementów z progrmowania np. zasotosowanie pętli "for", czy możliwosc deklarowania zmiennych.

Najpopolarnieszymi preprocesorami sa:

sass

less

Osobiście używam SASS'a, bo najczesciej jest uzywane we szelkiego rodzaju szablonach. Tego preprocesora róznież użyłem piszac tą strone

JavaScript natomiast zyskał przez ostatnie lata ogroną popolarność developerów, zatem technologi jest ogrom. W tym momencie najpolularniejszymi frameworkami (biblioteki rozszerzajace i ułątiwjące prace w Javascripcie) są:

vue

react

angular

W tym momencie wolny czasie poświęcam sie na nauce pierwszego z nich, bo daje duzo moliwosc manipolowanie elementami stroy i tworzyc komponenty z którch można "skleic" całą strone internetową.

Czy to już nie hobby?

Tworzenie stron jest dla mnie bardzo milą forma spedzania czasu wolengo. Cały czas przeglądam rózne fora internetowe w poszukiwaniu inspiracji oraz do poszerzania wiedzy na temat tej dziedziny. Poprzez takie działa ucze sie nowych rzeczy ale tez rozwijam kreatywnośc i wrażliwość na sztuke, bo strony internetowe to też dzieła sztuki (przykład)

W przyszłosći podjąłbym sie pracy webdevelopera, mysle ze moje hobby powoli bedzie stawało sie moją pracą.
